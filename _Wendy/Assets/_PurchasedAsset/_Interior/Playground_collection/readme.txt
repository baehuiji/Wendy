This package contains 17 different types of playground objects.
Just drag the prefabs into your scene and arrange them as you like.

You may also want to have a look at the example scene that contains all the prefabs.
Textures and normal maps are included.

All single parts using about 150-4500 Polygons.

----------------------------------------------------------------
2018-10-22:

Package has been updated to Unity 2018 and Maskmaps added.
These can be used for Unity Standard-Shader and HDRP-Lit-Shader.
